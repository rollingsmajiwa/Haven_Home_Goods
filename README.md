Haven Home Goods
---
### 1. Project Brief
The **Haven Home Good** website is a modern, high-end e-commerce platform designed for handcrafted furniture and home decor. It provides a streamlined, aesthetically pleasing interface where users can browse curated collections and manage their interior design with ease.


### 2. Business Rationale

In a competitive luxury home market, clarity and professionalism are key. This project addresses two primary business needs.

- Brand Authority: A clean, semantic design featuring glassmorphism and modern typography establishes trust and positions the platform as the premium provider.

- Data Collection: The structured product grid ensures that product details, pricing and customer ratings are captured correctly for seamless shopping experience.

### 3. Technologies used

- HTML5: Semantic structure for SEO and accessibility.
- CSS3: For responsive layout and modern styling (linked via `styles.css`).
- Git/Github: version control and feature-based branching.
- Lucide icons:Integrated for high-quality professional vector icons.
### 4. Accessibility features

- Semantic HTML: Using header, main, section and article tags for screen readers.
- List structures: Key features are organized in `<ul>` and `<li>` tags, providing a clear reading path for assistive technologies.


### 5. Product features
- **Hero Section**: A full-width banner with a clear Call to Action.
- **Glassmorphism UI**: Frosted glass effects on the header and navigation for a modern feel.
- **Dynamic Product Grid**: A responsive 3-column layout featuring items like fabric sofas and artisan ceramic vases.
